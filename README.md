# 🌺 Ask Isha — AI Portfolio Experience

> *Most portfolios tell you what someone has done. This one lets you ask.*

**[🚀 Live Demo →](https://ishapanchal-hue.github.io/ask-isha/)**

## ✨ What is this?

**Ask Isha** is not a resume. It's not a portfolio template. It's not an FAQ page.

It's a fully interactive, AI-powered personal experience — built to feel like a conversation with a real person. You ask questions, it answers. With photos, project demos, architecture diagrams, tech stacks, leadership stories, and everything in between.

Think **Linear** meets **Notion AI** meets **personal brand**.

## 🎯 Features

| Feature | Description |
|---|---|
| 💬 **Conversational UI** | Ask anything — typed or clicked |
| 🧠 **Smart Question Matching** | Type naturally — it understands paraphrases |

| 🌺 **Easter Eggs** | Explore and find them |


## 🚀 Projects Featured

### 🌊 ARGO AI Platform — *Winner @ KJSIT-IET-INTECH 2K26*
AI-Driven Conversational Intelligence Platform for ARGO Oceanographic Data. Natural language → SQL with 90%+ accuracy. Built with React, FastAPI, PostgreSQL, Gemini API, Docker.
> [Live Demo](https://nemoai.vercel.app/dashboard) ·

### 💊 Prescription Scanner
Healthcare OCR system using TrOCR (Microsoft Transformer) + fuzzy matching to digitize handwritten prescriptions into structured medicine carts.
> Python · FastAPI · PyTorch

### 📦 AI Inventory & Demand Intelligence — *SIH 2025 Shortlisted*
Demand forecasting, expiry-risk prediction, and automated smart discounting for perishables. Team: The Outliers.
> Next.js · FastAPI · TensorFlow · PostgreSQL

### 🌱 SmartGrow — *Published @ ICCCNT*
IoT-based automated irrigation system. Demonstrated ~18% water savings. Real-time monitoring dashboard.
> ESP8266 · Django · DHT11 Sensors
